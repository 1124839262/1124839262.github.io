关于 Hexo

今天，我刚刚搭建好了这个由 Hexo 驱动的个人博客。Hexo 是一个快速、简洁且高效的博客框架，能够将 Markdown 文件快速渲染成静态网页，让我可以专注于内容创作本身。

为什么选择 Hexo？

  • 高效快速：生成静态文件，网站加载速度飞快。
  • 主题丰富：拥有众多精美的主题可供选择，让博客外观轻松定制。
  • Markdown 支持：使用 Markdown 语法写作，非常方便。
  • 易于部署：可以轻松地部署到 GitHub Pages 等平台，免费托管。
