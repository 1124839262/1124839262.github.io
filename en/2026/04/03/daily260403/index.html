The project uses Gateway for routing, so I studied Spring Cloud Gateway.

Gateway is a routing framework.
spring-cloud-starter-gateway

  • Combined with Hystrix as a circuit breaker for avalanche prevention
  • Combined with Ribbon for load balancing
  • Combined with Redis for rate limiting
  • Combined with Actuator for monitoring
  • Combined with Nacos for service discovery

Our current project stack: JDK 8, Spring Boot Hoxton.SR12, Gateway 3.1.3, Hystrix 2.2.9.RELEASE, Ribbon 2.2.9.RELEASE, Nacos 2.2.5.RELEASE + OpenFeign 2.2.9.RELEASE

  • Hystrix can be replaced with Sentinel
  • Ribbon can be replaced with Spring Cloud LoadBalancer

If I were responsible for the architecture in the future:

JDK 8 + Spring Boot 2.7.18 + Spring Cloud 2021.0.9 + Spring Cloud Gateway 3.1.9 + Nacos 2.3.0 + Spring Cloud LoadBalancer 3.1.9 + Spring Cloud Sentinel 2021.0.1.0 + OpenFeign 3.1.9
