Spring AI Learning Notes

Spring AI is a framework for integrating AI capabilities into Spring applications. It provides a unified API for various AI services including chat models, embedding models, image generation, and more.

Core Concepts

  • Chat Model: Interface for conversational AI (e.g., OpenAI, Ollama)
  • Embedding Model: Converts text to vector representations
  • Vector Store: Stores and retrieves vector embeddings
  • RAG (Retrieval-Augmented Generation): Combines retrieval with generation for context-aware responses
  • MCP (Model Context Protocol): Standardized protocol for AI model context management
